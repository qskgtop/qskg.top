矿工常见问题解答

我们将不断整理常见问题，供大家参考。

新晋问题

1. 怎么设置部分卡挖xx，另一部分卡挖yy？

答：设置好挖xx的钱包和内核 → 高级设置-工作显卡选择挖xx的卡 → 开始挖矿 → 停止挖矿 → 桌面的“轻松原版”改名为“挖xx” → 回到软件 → 设置挖yy的钱包和内核，内核尽量不要跟挖xx的一样 → 高级设置 - 工作显卡选择挖yy的卡 → 开始挖矿 → 双击桌面的“挖xx”

2. 为什么N张卡，只有N-1张卡工作？

答：首先请到设备管理器中查看显示设备是否都正常，如果设备管理中都没有设备，说明没有接好，再看有无感叹号；然后到软件的高级设置，看看工作显卡是否有选择，正常情况不选择。另外teamredminer内核不支持n卡，如果混插时选择了trm内核，也会少卡。 另外，卡托延长线故障也可能卡，表现为挖着挖着少，重启电脑后正常，工作一阵子又少。

3. 能挖BTC、LTC、FIL、CKB、狗狗币吗？

答：不能，也不可能，因为体系不一样。最早CKB是可以显卡挖，现在已是ASIC的菜了。

4. 能隐藏界面吗？能开发老板键吗？

答：不能，也没有计划添加这个功能

5. 部分N卡停止挖矿死机

答：请使用4.2.8或以上版本，勾选高级设置最后面的“停止挖矿死机保护”，4.2.8版本可重新下载安装或通过群控一键升级矿工端时选择版本号

6. 带AMD集显的笔记本，如果只设置N卡工作？

答：内核不同，参数不同，使用推荐的nbminer内核时，高级设置附加参数填写-platform 1即可只使用N卡工作。

7. PhoenixMiner内核提示Allocating buffers failed with : clCreateBuffer (-61)

答： 最近Windows系统风控推送升级AMD驱动。请检查驱动是否被自动升级，软件左下角有显示驱动版本，重新安装回20.4.2或之前使用的版本即可。

8. PhoenixMiner内核提示Allocating buffers failed with : clCreateBuffer (-4)

答：2G物理内存或虚拟内存不够，按单卡6G的量设置，8张A卡设置48G虚拟内存。

驱动篇(A卡)

1. 460/470/480/560/570/580 推荐用什么驱动？

答：推荐18.12.3或者20.4.2，严重不推荐20.5.+，在极端情况可能蓝屏，例如：claymore15.0内核启用时序时。

2. 590推荐用什么驱动？

答：推荐20.4.2，严重不推荐20.5.+，在极端情况可能蓝屏，例如：claymore15.0内核启用时序时。

3. 5500XT/5600XT/5700/5700XT推荐用什么驱动？

答：新卡新驱动，可以考虑官方最新驱动，如果你无法拿捏，推荐20.4.2。

驱动篇(N卡)

1. 无法定位程序输入点 fatBinaryCtl.dll 于动态链接库

答：可能因停电或直接关机导致驱动损坏，重新安装驱动即可，驱动版本不限。

2. 1660s用什么驱动？

答：用最新驱动，下载www.nvidia.cn。

3. P106/P104/1066/1070/1070ti/1080/1080ti用什么驱动？

答：如无特别嗜好，不建议老卡追新，推荐399.24。

4. 为什么不显示温度、转速和功耗？

答：停止挖矿后删除C:Program FilesNVIDIA CorporationNVSMI vml.dll文件，再开始挖矿。

硬件篇

1. A卡和N卡可以混装吗？588 578 584 574 能混装吗？

答：可以任意搭配，混插显卡建议在“超频设置”小窗口右上角进行“单卡超频”。

2. H81A主板设置集显输出后，接6卡无法正常启动

答：DVMT Pre-Allocated内存设置为128或者更小，如下图。

3. 为什么GPU0算力偏低？

答：同时作为显示输出，可能算力低、甚至不稳定，建议到主板BIOS设置集显输出，教程。

软件篇

1. 为什么会“内核重启”？

答：虚拟内存不够、钱包地址错误、卡故障、延长线故障、超频参数导致的不稳定。

2. 用轻松矿工挖不了，用轻松原版测试就正常工作了，是怎么回事？

答：因为轻松原版不超频所以可以正常工作，100%确定是超频参数引起的，请调整超频参数。

3. 软件可以屏蔽某张卡吗？

答：可以，点软件高级设置，工作显卡选择要工作的显卡。

4. 为什么打开计算模式蓝屏？

答：某张卡的卡托或延长线故障，甚至卡故障。

BIOS和超频

1. N卡需要刷BIOS吗？

答：通常不刷，目前就P104需要刷BIOS改8G，详见教程。

2. 刷过BIOS的A卡怎么超频？

答：视BIOS质量而定，BIOS质量不好，需要通过软件超频达到最佳效果。

3. 原版BIOS的A卡可以直接挖吗？

答：可以！设置方法参考教程，该教程适用所有A卡原版BIOS挖矿，需提醒的是8g卡内核选claymore 15.0，4g卡选择PhoenixMiner。

4. 为什么超频提示温度失败？

答：设置的温度太高，如果设置的是95建议尝试90，如果设置的是90建议尝试85，如果设置的是85建议尝试80。

5. 为什么安装驱动提示DCH错误？

由于历史原因，nVidia同一个版本的驱动，有DCH和非DCH版本之分。 这个提示是现在安装的DCH版本，没法覆盖现在系统中的非DCH版本。 处理办法:
1，要么也按照非DCH版本
2，要么DDU卸载赶紧，安装DCH版本

WIN7

1. 所有的A卡都不推荐使用WIN7系统

2. N卡也尽量不使用WIN7系统，但是部分3G、4G、5G显存的N卡有时必须使用WIN7或OS

3. 原版WIN7系统安装驱动后，软件左下角驱动版本显示0.00以及设备管理器中感叹号，需要安装这个补丁
